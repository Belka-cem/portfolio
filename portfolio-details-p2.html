Site bruraliste pour un festival

Dans le cadre d’un cours de web-design nous devions imaginer un site pour présenter un festival dans le monde du numérique. Ce site doit être désigné avec le style brutaliste. Le site est composé de 4 pages: Accueil, programmation, infos pratiques, contact.

Programme

Comme vous pouvez le voir la couleur dominante est un bleu très aggressif, j'ai choisi cette couleur pour vraiment aller à contrario de ce que pouver proposer les sites classiques de festivale. C'est également le cas pour le placement des informations, en effet la navbar placé sur les cotés donne un effet vraiment pertubant, c'est voulue pour rester dans le thème brutaliste.
Ces maquettes m'ont permis d'avancer très rapidement car tous les choix esthétiques étaient choisies, il ne me restait plus qu'àme concentrer sur la partie création du site.

Conclusion

Ce projet m'a permis de maitriser l'outils Bootstrap ainsi que de renforcer les compétences que j'ai en HTML et CSS. Il m'a également permis de créer un site dans un style que je ne connaissais pas et de comprendre toute la difficulté à créer un site intuitif et ergononique rentrant dans les codes du brutalisme design.
